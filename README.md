# 🚦 IntelliTraffic AI

### Smart Traffic Monitoring & Signal Optimization Platform

IntelliTraffic AI is an advanced AI-powered traffic monitoring and signal optimization system built using Python, YOLOv8, OpenCV, Streamlit, Machine Learning, SQLite, and Google Maps integration.

The platform helps monitor traffic conditions, detect vehicles, estimate congestion levels, optimize signal timings, generate reports, and provide intelligent recommendations for traffic management.

---

## 🎯 Project Objectives

* Detect and count vehicles using AI
* Analyze traffic density in real time
* Optimize traffic signal timings
* Support multi-camera monitoring
* Monitor CCTV / RTSP traffic streams
* Predict future traffic volume using Machine Learning
* Generate PDF, CSV, and JSON reports
* Store traffic analytics in SQLite database
* Collect user feedback and reviews
* Provide smart recommendations for traffic operators

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/IntelliTraffic-AI.git
cd IntelliTraffic-AI
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run application:

```bash
streamlit run app.py
```
